# ビジョン: CalcEngine 関数電卓 API

## エグゼクティブサマリー

CalcEngineは、開発者が数式を文字列として送信し、正確な計算結果を受け取ることができるREST APIです。各チームが独自の数式パーサーや三角関数を実装する代わりに、CalcEngineのAPIを呼び出すだけで済みます。無料プランで普及を促進し、大容量利用向けの有料プランを提供するサブスクリプション型サービスとして提供します。

## スコープ内の機能（MVP）

- 数式評価: `"2 * sin(pi/4) + sqrt(16)"` のような文字列を受け取り、数値結果を返す
- 基本算術: 加算、減算、乗算、除算、べき乗、平方根、剰余、絶対値、床関数、天井関数、丸め
- 三角関数: sin、cos、tan、asin、acos、atan、atan2（度数モードおよびラジアンモード）
- 対数関数: 常用対数、自然対数、任意底の対数、exp
- 基本統計: 平均、中央値、最頻値、標準偏差、分散、最小値、最大値、合計、パーセンタイル（配列を受け付ける）
- 数学定数: pi、e、phi、sqrt(2)
- 組み合わせ論: 階乗、順列（nPr）、組み合わせ（nCr）
- エラー処理: ゼロ除算、定義域エラー（負の数の対数）、オーバーフロー、不正な数式に対する明確なエラーコード
- 無料プラン（月10,000回）および有料プランを含むAPIキー認証
- インタラクティブなサンドボックスとコード例を備えたAPIドキュメントポータル

## スコープ外の機能（MVP）

- 任意精度演算（フェーズ2）
- 行列・線形代数（フェーズ2）
- 微積分――微分、積分（フェーズ2）
- 金融数学――返済計画、NPV、IRR（フェーズ2）
- Python/JS/Java向けクライアントSDK（フェーズ2――MVPではHTTP直接呼び出しで十分）
- 解法のステップバイステップ解説（フェーズ3）
- 単位変換および物理定数（フェーズ3）
- バッチ処理・非同期Webhook（フェーズ3）
- 記号計算（フェーズ3）
- オンプレミスデプロイ（フェーズ3以降）

## 対象ユーザー

- 製品に数学機能が必要だが、自社での実装・保守を望まないアプリケーション開発者
- 学習者向けツールに計算バックエンドを必要とするEdTech企業
- 監査可能な計算を必要とするFinTechスタートアップ（有料プラン、フェーズ2重点）
