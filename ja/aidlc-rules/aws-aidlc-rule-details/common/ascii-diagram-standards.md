# ASCII図の標準規格

## 必須：基本ASCIIのみを使用すること

**重要**: 図には必ず基本ASCII文字を使用してください（最大限の互換性を確保するため）。

### ✅ 使用可能：`+` `-` `|` `^` `v` `<` `>` および英数字テキスト

### ❌ 禁止：Unicodeの罫線文字
- 禁止：`┌` `─` `│` `└` `┐` `┘` `├` `┤` `┬` `┴` `┼` `▼` `▲` `►` `◄`
- 理由：フォントやプラットフォームによってレンダリングが一致しないため

## 標準的なASCII図のパターン

### 重要：文字幅のルール
**ボックス内のすべての行は、スペースを含めて必ず同じ文字数でなければなりません**

✅ 正しい例（すべての行が67文字）：
```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                        |
+---------------------------------------------------------------+
```

❌ 誤った例（幅が不揃い）：
```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                   |
+---------------------------------------------------------------+
```

### ボックスパターン
```
+-----------------------------------------------------+
|                                                     |
|              Calculator Application                 |
|                                                     |
|  Provides basic arithmetic operations for users     |
|  through a web-based interface                      |
|                                                     |
+-----------------------------------------------------+
```

```
+-------------------------------------------------------+
|              Web Server (PHP Runtime)                 |
|  +-------------------------------------------------+  |
|  |  index.php (Monolithic Application)             |  |
|  |  +-------------------------------------------+  |  |
|  |  |  HTML Template (View Layer)               |  |  |
|  |  |  - Form rendering                         |  |  |
|  |  |  - Result display                         |  |  |
|  |  +-------------------------------------------+  |  |
|  +-------------------------------------------------+  |
+-------------------------------------------------------+
```

### 矢印と接続線
```
+----------+
|  Source  |
+----------+
     |
     | HTTP POST
     v
+----------+
|  Target  |
+----------+
```

### 水平フロー
```
+-------+     +-------+     +-------+
| Step1 | --> | Step2 | --> | Step3 |
+-------+     +-------+     +-------+
```

```

+----------+

    v

+----------+
```

## 検証

図を作成する前に確認してください：
- [ ] 基本ASCIIのみ：`+` `-` `|` `^` `v` `<` `>`
- [ ] Unicodeの罫線文字を使用していないこと
- [ ] 位置合わせにはスペースを使用（タブは不可）
- [ ] 角は `+` を使用
- [ ] **ボックスのすべての行が同じ文字幅であること**（スペースを含めて文字数を確認）
- [ ] テスト：等幅フォントでコーナーが縦に揃っているか確認

## 代替手段

複雑な図には、Mermaidを使用してください（`content-validation.md` を参照）
